# MobileNet

TensorFlow.jsを使った画像分類デモアプリケーションです。カメラ入力の映像から対象物を高速に分類し、日本語の分類結果を表示します。

## デモ
https://code4fukui.github.io/MobileNet/

## 機能
- カメラ入力を使った画像分類
- MobileNetモデルを用いた高速な分類
- 分類結果の日本語表示

## 必要環境
- WebブラウザでのJavaScript実行環境

## 使い方
ブラウザで上記デモページを開くと、カメラ入力に対する分類結果が表示されます。
